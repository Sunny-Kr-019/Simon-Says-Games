# About game -

The Simon Says Game is a fun and interactive memory-based game where players must follow a sequence of colors and sounds that gradually increase in complexity. Inspired by the classic "Simon Says," this game tests concentration and pattern recognition skills. With each round, a new color is added to the sequence, challenging players to recall and repeat the pattern correctly. Built using HTML, CSS, and JavaScript, the game features engaging visuals, sound effects, and an intuitive interface. It offers an enjoyable way to improve memory while having fun.
